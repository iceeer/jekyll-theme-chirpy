0) 安装 Visual Studio 2008 QFE 补丁，修正下载源代码时的性能问题。

1) 打开Visual Studio 2008，进入 Tools > Options > Debugging > General（工具>选项>调试>常规）.

  • 关闭”Enable Just My Code”(启用“仅我的代码”) 设置
  • 开启”Enable Source Server Support” (从符号服务器加载符号)设置

2) 进入”Symbols”页设置下载源码的路径

  • 设置符号文件路径: http://referencesource.microsoft.com/symbols
  • 设置缓冲文件夹“Cache symbols from symbols server to this directory”（将符号从符号服务器缓冲到此目录）路径
  • “Search the above locations only when symbols are loaded manually“(使用以上位置加载符号)选项：如果启用则手动下载源码

-如果使用Visual Stdio 2005 ，就看这篇 。

-相关工具:

  • Net Mass Downloader - 把全部源码下载下来的工具

也可以自己去微软官方下载提供的源码库

  • Visual Studio Team System 2008 Team Suite 90天评估版 - 这可是最重要的，无需介绍吧
