修改php.ini确认有这行：extension=php_ldap.dll
修改config_inc.php：
# SET UP the Login Method
$g_login_method = LDAP;
$g_ldap_server = 'ldap://10.136.238.54:389';
$g_ldap_root_dn = 'O=STD';
$g_ldap_organization = '';
#Domino LDAP使用cn，Mantis不支持中文名登录
$g_ldap_uid_field = 'uid';
$g_ldap_bind_dn ='';
$g_ldap_bind_passwd = '';
# Use email address in LDAP rather than the email stored in the database.
$g_use_ldap_email = ON;
# The LDAP Protocol Version, if 0, then the protocol version is not set.
$g_ldap_protocol_version = 3;
# --- signup ----------------------
$g_allow_signup = OFF;
$g_lost_password_feature = OFF;

*注意原来系统里的管理员帐号不要删掉，LDAP里要有同名的管理员帐号才能登录
