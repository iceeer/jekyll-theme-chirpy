快捷方式
"C:Program FilesCommon FilesVMwareVMware Remote Console Plug-invmware-vmrc.exe" -h 127.0.0.1:8333 -M 64
其中"-M"后的参数managed object ID 是对应虚拟机的编号，请自行打开参阅 %ALLUSERSPROFILE%Application DataVMwareVMware ServerhostdvmInventory.xml
启动虚拟机时出错You have started VMware Server with an out-of-date copy of the application. VMware Server cannot power on virtual machines using this copy (version 2.0.2). To power on this virtual machine, run VMware Server from the default location.
修改虚拟机配置文件 vmx.noUIBuildNumberCheck = "TRUE"
