标识类型 使用 包含数据库信息 各数据库 文档修改 数据库副本 Notes数据库复制 系统数据库复制
Database ID 数据库标识 唯一 n/a 不同 不同 不同
Replica ID 数据库副本标识 唯一 n/a 相同 different same
UNID (Universal Note ID) 唯一地确定了文档 唯一 相同 相同 相同 相同
OID(Originator ID) 确定文档(note)的特殊的修订版本 唯一 different same same same (w/o revision)
NID (Note ID) 确定给定数据库中的一个特殊的文档 不唯一 相同 不同 相同 相同
GNID (Global Note ID) 确定一个特殊数据库中的一个特殊文档 不包含 唯一 same different different same
IID (Instance ID) 确定一个给定数据库中的一个文档(note)的特殊修订版本 不包含 不唯一 different different different same (w/o revision)
GIID (Global Instance ID) 确定一个特殊数据库中的一个文档(note)的特殊修订版本 包含 唯一 different different different same (w/o revision)

第一行：OF后为文件号(File)

第二行：ON后为文档号(Note)

第三行：序列时间（SD后）和序列号（SN后）

第四行：DB后16位数字与数据库副本标识相同

第五行：NT后为Note ID

第一、二行组合为Universal Note ID

第一、二、三行组合为Originator ID

第四、五行组合为Global Note ID
